## Summary

This session focused on Active Directory Certificate Services deployment and adding a new Linux VM to the lab infrastructure. The domain controller was configured as a Certificate Authority, creating the necessary security groups and memberships. A new Ubuntu-based ticketing system VM was deployed to Azure with its own virtual network and basic network security rules.

- Created TICKET-vnet virtual network

  - HTTP (TCP port 80, priority 320)

## Notes

The TICKET01 NSG allows SSH and HTTP from any source address (*). This is appropriate for a lab environment but should be restricted if the VM handles sensitive data or moves toward production use.
